## Dependencies installation

```shell
poetry install --without dev
```

## Development

### Development dependencies installation

```shell
poetry install --with dev
```

### Pre-commit  hooks

Install pre-commit hooks

```shell
pre-commit install
```

Run manually

```shell
pre-commit run --all-files --color always --verbose
```
